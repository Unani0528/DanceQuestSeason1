# 🧹 데이터 조작 및 전처리 (Data Manipulation & Preprocessing)
**데이터 조작 및 전처리(Data Manipulation & Preprocessing)**  
AI 학습 모델에 투입되기 전, 원시 데이터(Raw Data)를 분석하고 모델이 학습할 수 있는 깨끗하고 유용한 형태로 변환하는 과정

## 데이터 조작 및 전처리의 개념

| 개념 | 설명 |
| :--- | :--- |
| **데이터 조작 (Manipulation)** | 데이터를 분석 목적에 맞게 선택, 결합, 변형, 집계하는 과정<br>주로 **Pandas** 라이브러리를 사용하여 DataFrame의 형태로 데이터를 가공 |
| **데이터 전처리 (Preprocessing)** | 데이터의 품질을 개선하고, 불완전하거나 일관성 없는 데이터를 수정하여 모델 학습에 적합한 상태로 만드는 과정<br>주로 **결측치/이상치 처리** 및 **특성 스케일링**을 포함 |

---

##  데이터 전처리의 핵심 기법

전처리는 데이터를 깨끗하게 만들고, 모델이 이해할 수 있는 형태로 변환

### 데이터 클리닝 (Data Cleaning)

| 기법 | 설명 |
| :--- | :--- |
| **결측치 처리 (Missing Values)** | 비어있는 값(`NaN`, `None`)을 제거하거나 다른값으로 대체함. |
| **이상치 처리 (Outliers)** | 다른 값과 동떨어져 있는 데이터를 수정하거나 제거함. |
| **오류 및 중복 처리** | 입력 오류(오타)나 완전히 동일한 중복 데이터를 처리함. |

### 데이터 변환 및 스케일링 (Data Transformation & Scaling)

| 기법 | 설명 |
| :--- | :--- |
| **특성 스케일링 (Scaling)** | 변수들의 값 범위를 일정하게 맞춰 모델 학습의 안정성을 높임. |
| **표준화 (Standardization)** | 데이터를 평균 0, 표준편차 1인 분포로 변환함. |
| **정규화 (Normalization)** | 데이터를 0과 1 사이의 값으로 변환함. |

### 특성 인코딩 및 공학 (Encoding & Feature Engineering)

| 기법 | 설명 |
| :--- | :--- |
| **범주형 인코딩** | '성별', '직업' 등 문자로 된 범주형 데이터를 모델이 이해할 수 있는 숫자로 변환함. | 
| **원-핫 인코딩** | 범주 수만큼 새로운 이진(0 또는 1) 특성을 만듦.| 
| **특성 공학 (Feature Engineering)** | 기존 특성을 조합하거나 변형하여 모델 성능을 높일 수 있는 새로운 특성을 만듦. |

---

## 주요 라이브러리 및 역할

| 라이브러리 | 주요 역할 |
| :--- | :--- |
| **Pandas** | 데이터 불러오기, 조작, 정렬, 그룹화, **결측치 처리** (기반) |
| **NumPy** | 고성능 배열 기반 계산, **수학적 연산** 지원 (Pandas 내부 연산 기반) |
| **Scikit-learn** | **전처리 모듈** 제공 (`StandardScaler`, `OneHotEncoder`), ML 모델링 |

---

## 머신러닝 학습 과정 개요

- 원시 데이터를 데이터프레임으로 변환해 기본 통계와 분포를 시각화해서 이해  
- 결측치, 이상치 등 데이터 정제를 수행  
- 특성 스케일링 및 인코딩 등 전처리 적용  
- 학습용, 검증용, 테스트용 데이터로 분할 (클래스 비율 유지)  
- 모델 선택 후 학습 진행 (예: 로지스틱 회귀, 랜덤 포레스트, SVM 등)  
- 테스트 데이터로 예측 수행 후 정확도, 분류 리포트, 혼동 행렬로 평가 및 시각화
