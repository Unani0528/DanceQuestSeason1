## 1. 데이터 조작 및 전처리 (Data Manipulation & Preprocessing)

**데이터 조작 및 전처리(Data Manipulation & Preprocessing)**는 AI 학습의 출발점입니다.  
원시 데이터를 분석하고 모델이 정확히 학습할 수 있도록 결측치 처리, 이상치 수정, 스케일링 등 다양한 과정을 거쳐 데이터를 정제하는 단계입니다.

### 주요 학습 내용

- 데이터 조작: Pandas를 활용해 데이터 선택, 변형, 집계 등 데이터 구조를 다루는 기술  
- 데이터 전처리: 결측치와 이상치 처리, 특성 스케일링(표준화, 정규화)  
- 특성 인코딩: 범주형 데이터를 숫자형으로 변환 (원-핫 인코딩 등)  
- 특성 공학: 모델 성능 향상을 위한 새로운 기능(feature) 생성  

### 중요 라이브러리

| 라이브러리 | 역할 |
| :--- | :--- |
| Pandas | 데이터 로드, 조작, 결측치 처리 등 |
| NumPy | 수치 계산과 배열 처리 |
| Scikit-learn | 데이터 전처리 도구 제공 및 ML 모델 구현 |

## 2. 머신러닝 (Machine Learning)

머신러닝은 전통적인 알고리즘을 활용해 데이터를 기반으로 패턴을 학습하는 영역입니다.  
지도학습, 비지도학습 등 다양한 알고리즘을 다루며, 데이터 준비 후 모델링, 하이퍼파라미터 튜닝, 평가 등을 포함합니다.

### 주요 학습 내용

- 지도학습 알고리즘: 로지스틱 회귀, 의사결정나무, SVM, 랜덤포레스트 등  
- 데이터 분할: 학습/검증/테스트 세트 생성 및 클래스 분포 유지  
- 모델 평가: 정확도, 정밀도, 재현율, F1-score, 혼동 행렬  
- 하이퍼파라미터 조정과 과적합 방지 기법  

---

## 3. 딥러닝 (Deep Learning)

딥러닝은 인공신경망을 기반으로 한 고도화된 머신러닝 분야입니다.  
복잡한 패턴 인식 및 대규모 데이터 처리가 가능하며, 이미지, 음성, 자연어 처리에 주로 활용됩니다.

### 주요 학습 내용

- 기본 신경망 구조: 퍼셉트론, 다층 퍼셉트론(MLP)  
- 합성곱 신경망(CNN): 이미지 분석을 위한 핵심 구조  
- 순환 신경망(RNN), LSTM: 시계열/텍스트 데이터 학습 모델  
- 최신 모델: 트랜스포머, 어텐션 메커니즘  
- 딥러닝 프레임워크: TensorFlow, Keras, PyTorch  
- 과적합 방지: 드롭아웃, 정규화  
- 모델 평가 및 시각화  

### 실습 예제

- MNIST 숫자 이미지 분류 (CNN)  
- AlexNet 변형 모델 구현과 학습
